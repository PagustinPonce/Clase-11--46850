Comandos basicos de git

cuando querramos ignorar un archivo no tiene que estar incluido ya
1. git init
Este comando inicia nuestros comandos

2.git status
para saber la situacion actual de nuestro proyecto

3.git add .
para incluir archivos

4.git commit -m ""
con este confirmamos todos los cambios que hermos hecho

5.git log
genera un historial completo de los ultimos commits, usuario, fecha y hora de esos cambios

6.git log --online
sirve para ver el historial de manera mas corta

7.git branch
muestra las ramas disponibles

8. git branch nombre cualquiera
crea una rama con el nombre que le asignen

9. git checkout
sirve para cambiar entre las ramas disponibles.
